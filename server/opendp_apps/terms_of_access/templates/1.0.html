  1. OVERVIEW

OpenDP is a community effort led by researchers at Harvard University that provides Users with a system of tools for enabling privacy-protective analysis of sensitive personal data. The target use cases for OpenDP are to enable government, industry, and academic institutions to safely and confidently share sensitive data to support scientifically oriented research and exploration in the public interest.

The OpenDP Application, DPcreator (Application), uses the OpenDP library of algorithms to generate differentially private statistical releases. These algorithms inject statistical noise into a dataset to protect individual information – the more statistical noise that is injected, the more privacy is ensured. However, it is important to note that the injection of statistical noise also decreases the accuracy of the data – thus, the calculation of an efficient privacy budget, measured by the variable epsilon, is extremely important to differential privacy.

Users will be able to upload a dataset to the Application from a third party platform. By analyzing the metadata of that dataset and integrating user inputs, the Application will guide the user in generating an epsilon value and constructing statistics which will be returned as a differentially private (DP) Release. This DP Release, which conforms to a privacy budget and is constructed by OpenDP algorithms as well as user inputs, is then returned to the User.
